✦ Behind The "Digital Imaging for Cartier WGTA0067" Design ✦

In this project, I made a digital imaging of a box from a cartier brand watch. In the original photo, there is only a watch
and making a box from this watch is my task. I deliberately chose the dark red color as the box and the basis of the visual
elements because it was taken from the original box which was that color, with this combination of red and gold, it made a
luxurious and calm impression. So it can be concluded that there are 3 main visual elements in this image manipulation.
These visual elements are the original image of the watch, the watch box, and the red fabric.
